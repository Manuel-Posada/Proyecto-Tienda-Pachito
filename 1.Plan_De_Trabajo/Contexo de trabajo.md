# 🌍 Contexto de Trabajo

## 🔹 Descripción General

El proyecto consiste en el desarrollo de un sistema básico de gestión para la tienda local **Pachito**, con el objetivo de mejorar el control de productos, ventas e inventario.

Actualmente, muchos de estos procesos se realizan de forma manual, lo que puede generar errores en el registro de precios, pérdidas de información y dificultades para conocer el estado real del negocio.

---

## 🔹 Problema a Resolver

La tienda presenta las siguientes dificultades:

- Falta de control preciso del inventario  
- Errores en el registro de ventas  
- Dificultad para conocer ingresos diarios  
- Procesos manuales que consumen tiempo  

Estas problemáticas afectan la eficiencia del negocio y la toma de decisiones.

---

## 🔹 Solución Propuesta

Se desarrollará un sistema web que permita:

- Registrar productos con su información básica  
- Gestionar ventas de forma rápida y automática  
- Actualizar el stock en tiempo real  
- Generar reportes de ventas  

El sistema estará diseñado para ser fácil de usar y adaptado a las necesidades del negocio.

---

## 🔹 Entorno de Desarrollo

El proyecto será desarrollado utilizando:

- **Django** para el backend  
- **Angular** para el frontend  
- **MySQL** como sistema de base de datos  

El equipo trabajará bajo la metodología **Scrum**, organizando el desarrollo en sprints para entregar funcionalidades de manera incremental.

---

## 🔹 Alcance del Proyecto

El sistema incluirá funcionalidades como:

- Registro de productos  
- Registro de ventas  
- Actualización de inventario  
- Registro de compras  
- Generación de reportes básicos  
- Alertas de stock bajo  

No se incluirán funcionalidades como gestión de clientes, ya que no aportan valor inmediato al negocio en esta etapa.

---

## 🔹 Usuarios del Sistema

El sistema estará dirigido principalmente a:

- Dueño de la tienda  
- Empleados encargados de ventas  

Estos usuarios interactuarán con el sistema de forma diaria para gestionar las operaciones del negocio.

---

## 🎯 Objetivo

El objetivo principal es desarrollar una solución que:

- Reduzca errores humanos  
- Optimice el tiempo de trabajo  
- Mejore el control del inventario  
- Facilite la toma de decisiones  

De esta manera, se busca mejorar la eficiencia y organización de la tienda Pachito mediante el uso de tecnología.
